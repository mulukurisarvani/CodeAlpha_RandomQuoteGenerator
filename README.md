# CodeAlpha_RandomQuoteGenerator

A clean, minimalist Web Application built using pure HTML, CSS, and JavaScript that displays a dynamic, random motivational quote on every click or page refresh. This project was developed as part of my **Web & App Development Internship** at **CodeAlpha**.

---

## 🚀 Features
* **Instant Quotes:** Generates a fresh, inspiring quote dynamically every time the user hits the "New Quote" button.
* **Responsive Layout:** Completely optimized for viewing on laptops, tablets, and smartphones.
* **Smooth Visual Transitions:** Features a dynamic background color switcher utilizing an aesthetic pastel palette that blends seamlessly on changes.
* **Self-Contained Logic:** Engineered using clean DOM manipulation methodologies and local data arrays for maximum speed and offline support.

---

## 🛠️ Built With
* **HTML5:** Structuring the core minimalist container layout.
* **CSS3:** For modern Flexbox alignment, customized typography shadows, transitions, and hover behaviors.
* **JavaScript (ES6+):** Handling event listener interfaces, local cache state arrays, and mathematical random array indexing.
